Lors de ce stage de première année, j’ai eu l’opportunité de découvrir et de me perfectionner sur le framework Django, en Python. La mission principale consistait à créer un système d’archivage basé sur une base de données SQLite. Ces bases contiennent des données issues de plusieurs zones surveillées : un parking, un port et une piscine.

Une fois une zone sélectionnée, une seconde sélection permet de préciser l’emplacement exact au sein de cette zone.

Après cette sélection, les données les plus récentes s’affichent. Si la base de données est inaccessible, l’utilisateur est notifié via un bandeau visible sur la page.
